# Расчет размера выборки для трех метрик амилоидогенной нагрузки

Размер выборки был рассчитан для трех метрик, определенных на основе предсказаний амилоидогенности, экспрессии транскриптов и признаков белковой последовательности:

1. **Burden**: сумма `log2(TPM + 1)` для амилоидогенных protein-coding транскриптов/протеоформ.
2. **Fraction**: доля суммарного protein-coding сигнала `log2(TPM + 1)`, приходящаяся на амилоидогенные транскрипты/протеоформы.
3. **KDmax-weighted Fraction**: доля KDmax-взвешенного сигнала, где `log2(TPM + 1)` умножался на максимальный Kyte-Doolittle hydrophobicity score (`KDmax`), а затем нормировался на общий KDmax-взвешенный protein-coding сигнал.

Расчет выполнялся для сравнения baseline `AC/control` против baseline `TOL/chili`. Использовался двусторонний уровень значимости `alpha = 0.05`, целевая мощность 80% и 90%, двухвыборочная модель с равными размерами групп. Эффект оценивался по пилотным данным как стандартизированная разница между группами; для планирования использовался абсолютный Hedges' g.

## Основной расчет: все baseline samples

В полный baseline анализ вошли 6 образцов `AC/control` и 5 образцов `TOL/chili`.

| Метрика | Hedges' g | Welch p-value | n на группу для 80% мощности | n на группу для 90% мощности |
|---|---:|---:|---:|---:|
| Burden | 0.336 | 0.552 | 140 | 187 |
| Fraction | 0.603 | 0.281 | 45 | 59 |
| KDmax-weighted Fraction | 0.697 | 0.222 | 34 | 45 |

По полному baseline набору наиболее информативными для планирования оказались нормализованные метрики: **Fraction** и **KDmax-weighted Fraction**. Для подтверждающего исследования по метрике Fraction требуется примерно **45 образцов на группу** для 80% мощности и **59 образцов на группу** для 90% мощности. Для KDmax-weighted Fraction требуется примерно **34 образца на группу** для 80% мощности и **45 образцов на группу** для 90% мощности.

Абсолютная нагрузка (**Burden**) имеет более слабый стандартизированный эффект и требует значительно большего размера выборки: около **140 образцов на группу** для 80% мощности и **187 образцов на группу** для 90% мощности.

## Чувствительный QC-сценарий: Figure 3 filtered set

Дополнительно был рассчитан более строгий сценарий после QC-фильтрации, соответствующий Figure 3 analysis set. В него вошли 4 образца `AC/control` и 4 образца `TOL/chili`; были исключены образцы с низким protein-coding annotation coverage, отсутствием классифицированных protein-coding строк или выраженным низким AC outlier.

| Метрика | Hedges' g | Welch p-value | n на группу для 80% мощности | n на группу для 90% мощности |
|---|---:|---:|---:|---:|
| Burden | 0.273 | 0.673 | 212 | 284 |
| Fraction | 0.228 | 0.729 | 304 | 406 |
| KDmax-weighted Fraction | 0.141 | 0.828 | 787 | 1053 |

В этом строгом QC-наборе различия между группами почти исчезают, поэтому расчетный размер выборки становится очень большим. Этот результат показывает, что пилотные оценки чувствительны к выбору образцов и фильтрам качества.

## Практическая рекомендация

Для статьи и планирования следующего исследования наиболее рационально использовать **Fraction** и **KDmax-weighted Fraction** как основные нормализованные endpoints. На основании полного baseline анализа рекомендуемый размер выборки составляет:

- минимум **45-60 образцов на группу** для проверки нормализованной Fraction с мощностью 80-90%;
- примерно **35-45 образцов на группу** для KDmax-weighted Fraction с мощностью 80-90%;
- если Burden остается самостоятельным primary endpoint, потребуется существенно более крупная выборка, ориентировочно **140-190 образцов на группу**.

С учетом нестабильности пилотных оценок и малого исходного размера выборки консервативная рекомендация для валидационного исследования: планировать не менее **60 образцов на группу**, если primary endpoint является Fraction, или не менее **45 образцов на группу**, если primary endpoint является KDmax-weighted Fraction. Текущие данные следует рассматривать как пилотную оценку эффекта, а не как подтверждающее доказательство биомаркера.
